# Example Usage

forecast_usage_example.py:
```
from forecast import Forecast as forecast_module

api_key = '0cb78f21a858991925ec9f90bbcd78b5'
city = 'Chicago'
country = 'United States of America'

forecast = forecast_module(api_key=api_key, city=city, country=country, days=40)
forecast.fetch_forecast()
print(forecast.weather())
print(forecast.temperature())
print(forecast.humidity())
print(forecast.wind())
print(forecast.cloud_cover())
print(forecast.precipitation())

print()

forecast.city = 'Manhattan'
forecast.days = 2
forecast.fetch_forecast()

print(forecast.temperature())
print(forecast.humidity())
print(forecast.wind())
print(forecast.cloud_cover())
print(forecast.precipitation())

print()

forecast.city = 'New York'
forecast.days = [2, 5, 9]

    {day(Counting from today): [datetime(Epoch Unix), Cloud Cover]}

precipitaion():
    {day(Counting from today): [datetime(Epoch Unix), Precipetation]}
```

# Usage
Forecast():
```
Creates Forecast() object

Arguments:
    api_key (str): OpenWeather.com api key
    city (str): city name
    country (str): country name OR ISO_3166-2 country code
    days (int) OR (list): integer for range of days 0 - days OR list for specific days [Default: 7]
```

fetch_forecast():
```
Fetches the forcast from the API with current provided arguments of city, country, and api_key

Arguments:
   None
```

weather():
```
Returns JSON data of weather descriptions, both a general and more detailed description along with a date/time

Arguments:
   None
```

temperature():
```
Returns JSON data of temperature forecast, average, min, and max along with a date/time

Arguments:
   None
```

humidity():
```
Returns JSON data of humidity forecast, humidy with a date/time

Arguments:
   None
```

wind():
```
Returns JSON data of wind forecast, speed and direction with a date/time

Arguments:
   None
```

cloud_cover():
```
Returns JSON data of cloud cover forecast, cloud cover with a date/time

Arguments:
   None
```

precipitation():
```
Returns JSON data of precipitation forecast, precipitation with a date/time

Arguments:
   None
```

request():
```
Returns a built API URL with parametrs for debuging purposes

Arguments:
   None
```

api_key:
```
(str): API key for OpenWeather.com
```

city:
```
(str): city name
```

country:
```
(str): country name, either full name or ISO-3166-2 country code
```

country_code (Read Only):
```
(str): ISO-3166-2 country code of provided country
```

days:
```
(int): range of days to be returned, from 0 - days [Default: 0-7]

OR

(list): list of days to be returned
```
